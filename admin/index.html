What is smash?

smash (SMS Aggregation & Sharing Hub) is an Android app that forwards SMS and MMS messages from a dedicated phone to your email and/or other phone numbers.

Perfect for managing a shared phone number (like a property management line) where multiple people need to see and respond to messages.

Quick Start

  1. Install smash on an Android phone
  2. Grant all permissions when prompted
  3. Set smash as the default SMS app
  4. Add yourself as a target: Cmd add +15551234567
  5. Configure email forwarding: Cmd setmail https://your-endpoint.com

See the full command reference for all options.

How It Works

When someone texts the smash phone:

  • The message is forwarded to all configured targets (phone numbers)
  • If email is configured, a copy is sent to your inbox
  • Targets can reply directly - smash will forward the reply back to the original sender

You control smash by sending SMS commands from a target phone. Commands start with a prefix (default: Cmd).
